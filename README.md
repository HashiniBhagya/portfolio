# Hashini Bhagya | Personal Portfolio

A modern, fully responsive portfolio website built with HTML5, CSS3, and JavaScript featuring dark/light theme toggle, smooth animations, and GitHub Pages CI/CD deployment.

## 🌟 Features

- Responsive Design (480px to 4K)
- Dark/Light Theme with localStorage
- Smooth Animations & Scroll-triggered Reveals
- Dynamic Typing Effect
- ContactForm with FormSubmit Integration
- Modern UI/UX with Glassmorphism
- SEO Ready & Accessible

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Icons:** Font Awesome 6.4.0
- **Fonts:** Space Grotesk
- **Deployment:** GitHub Pages
- **CI/CD:** GitHub Actions
- **Forms:** FormSubmit.co
